## Requires discord.py with aiohttp.
### .env file should include the following:
`BOT_TOKEN=`
`LOG_CHANNEL_ID=`
`ROLE_CHANNEL_ID=`
`CVE_CHANNEL_ID=`
`NVD_API_KEY=`

### Features:
- Linux distribution self-assignable roles with paginated interface
- Server audit logging (messages, members, roles, channels, voice)
- NVD CVE feed with Linux-relevant filtering and noise reduction
- Automatic duplicate prevention with persistent state

Distro list will be updated once in a blue moon.
Actively maintained.
